# QRandomPicker

QRandomPicker 是一个基于 Qt 的图形化工具，用于从多个名单中随机抽取姓名。
它支持公平模式和随机模式，并提供可视化的名单管理功能。

## 功能特性

* **多标签页管理**：每个标签页对应一个独立的名单。
* **名单编辑**：支持动态添加、删除姓名，实时保存。
* **两种抽取模式**：

  * **随机模式**：每次独立随机抽取。
  * **公平模式**：一轮内不重复抽取。
    注意：一次可能抽取到多个相同姓名

* **自定义抽取人数**：通过 `+` / `-` 按钮调整每次抽取的人数。
* **结果展示**：抽取结果在右侧区域以网格形式整齐排列，自动适应窗口大小

           注意：仅对长度少于5的中文名适用

## 编译依赖

Qt6

## 使用说明

1. **启动程序**：主窗口默认包含一个或多个标签页，每个标签页对应一个名单。
2. **切换名单**：点击标签页标题可切换不同名单。
3. **管理姓名**：

   * 在左侧列表中，可选中某个姓名，点击“删除选中”移除。
   * 点击“添加名字”弹出输入框，输入新姓名。

4. **选择模式**：

   * **随机模式**：每次抽取独立，结果可能重复。
   * **公平模式**：已抽取的姓名在左侧变为蓝色，当所有姓名均被抽过后，蓝色标记清除，开启新一轮。

5. **设置抽取人数**：通过 `-` 和 `+` 按钮调整本次抽取的人数。
6. **开始抽取**：点击“开始抽取”按钮，右侧区域显示抽取结果。
